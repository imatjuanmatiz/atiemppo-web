# SICETAC al Instante Web

## Asunto
SICETAC al Instante Web: una interfaz simple para una consulta compleja

## Preheader
La consulta de costos puede ser tecnica por dentro, pero no deberia sentirse pesada para quien la usa.

## Objetivo
Presentar la version web como una capa visual mas clara y mas alineada con la marca ATIEMPPO.

## CTA principal
Abrir SICETAC al Instante Web

## URL destino
https://sicealinstante.vercel.app/

## Cuerpo base
Las consultas de costos de transporte tienen una complejidad real detras. Pero eso no significa que la experiencia de uso tenga que sentirse compleja.

Esa es la apuesta de **SICETAC al Instante Web**.

Estamos construyendo una interfaz mas clara para consultar origen, destino, configuracion vehicular y carroceria sin exponer toda la complejidad interna del sistema. Nos interesa que se sienta como producto, no como formulario tecnico.

Detras de esa decision hay una idea sencilla: si una herramienta da confianza visual, la adopcion mejora. Y si la adopcion mejora, el valor analitico deja de quedarse encerrado en pocos usuarios.

Esta version web tambien conversa con el resto del ecosistema:

- se conecta con ATIEMPPO como marca
- vive al lado del canal de WhatsApp
- convive con el GPT y con la capa editorial

No queremos herramientas aisladas. Queremos una experiencia conectada.

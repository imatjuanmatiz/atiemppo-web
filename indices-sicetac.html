DE ARCHIVOS SUELTOS A UNA BASE UTIL

Tomamos los archivos mensuales de SICETAC, unificamos hojas, normalizamos variables y aplicamos una sola regla de consistencia temporal: cuando un periodo tenia varios cortes, se conservo el mas reciente. A partir de ahi construimos un universo comparable y una capa de trabajo lista para negocio, analisis y publicacion.

El universo de referencia se mantiene estable: carga General, unidad Estacas, condicion Cargado. Sobre esa estructura se pueden analizar configuraciones vehiculares, capitales de origen e indices por ruta sin rehacer el proceso cada vez.

Formato del activo: bases Excel, tablas consolidadas en DuckDB y documentos metodologicos listos para descarga o uso en proyectos a medida.

Cobertura

10 configuraciones, 28 capitales y miles de combinaciones listas para cruces por periodo, ruta y ciudad.

Lectura 2025

Serie base 202501 = 100 para ver trayectorias largas y comparar el comportamiento acumulado de los valores.

Lectura 2026

Serie base 202512 = 100 para leer mejor el pulso reciente del mercado y evitar una base demasiado lejana.
